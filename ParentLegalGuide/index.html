Legal Blueprint by Tier

🚀 LaunchPad Tier

  • Non‑commercial sandbox; no formal entity required
  • Educational license signed by parent/guardian
  • All contracts executed by parent only
  • Revenue capped below reporting thresholds
  • Legal Basis: Texas Family Code §151.001

🪐 Orbit Tier

  • UTMA Custodial LLC formation
  • Operating agreement with spending thresholds
  • Custodian signs contracts; minor signs internal resolutions
  • Legal Basis: Texas Property Code §141.001; Texas Business Organizations Code §3.101

🌌 Galaxy Tier

  • Trust overlay for staggered distributions
  • Advisory board charter and governance templates
  • Insurance and compliance automation
  • Legal Basis: Texas Property Code §141.007; Texas Trust Code §111.001

🌠 Nova Tier

  • LLC or Texas nonprofit formation
  • 501(c)(3) application and Form 990 filing
  • Legacy planning via trust and board succession
  • Legal Basis: Texas Business Organizations Code §22.001; IRS Form 1023
